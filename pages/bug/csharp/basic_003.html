　メモ帳から始めるC# その1で「なんか思ってたのと違うな」と違和感を覚えた方もいるかもしれません。

　メモ帳から始めるC# その1で作成したのは文字だけで動作するCUI[Character User Interface]という操作形体のアプリケーションでした。ウィンドウが表示され、中にテキストボックスやボタンがあるアプリケーションもそのようにソースコードを書くことで作成することができます。

　HelloWorld.cs と同じフォルダに HelloWorld2.cs というファイルを作り、下記のソースコードをそのまま貼り付けて保存してみてください。

　メモ帳から始めるC# その1の HelloWorld.cs と同様にコンパイルを行います。

　下記コマンドを実行するとコンパイルが実行され、 HelloWorld2.cs と同じフォルダに HelloWorld2.exe が作成されます。

　今回は『/t:winexe』という命令を追加しているので注意してください。

C:\Users\admin\Desktop\cs>C:\Windows\Microsoft.NET\Framework64\v4.0.30319\csc.exe /t:winexe HelloWorld2.cs

　作成された HelloWorld2.exe を実行すると HelloWorld.exe とは違い、テキストボックスとボタンがあるウィンドウが表示されたと思います。なんとなくイメージしていたものはこちらではなかったでしょうか？

　ちなみに文字だけの操作と操作の結果等が文字だけで返ってくるような操作形体をCUI[Character User Interface]と呼ぶのに対し、ウィンドウやボタン、テキストボックスなどマウスで動かすことのできるような操作形体をGUI[Graphical User Interface]と呼びます。
